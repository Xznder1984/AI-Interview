# OpenRouter API Key Guide

Get your free API key and start practicing interviews in minutes.

---

## Why Do I Need an API Key?

This application uses OpenRouter API to power realistic AI interviews. OpenRouter is a gateway to multiple high-quality AI models.

**You bring your own API key.** This means:

- ✅ You control your costs
- ✅ Your data stays in your browser
- ✅ Free tier available for testing
- ✅ No hidden fees or sign-up costs

---

## Getting Your API Key

### Step 1: Sign Up

Visit [OpenRouter.ai](https://openrouter.ai)

Click "Create Account" and sign up with:

- Email
- Google
- GitHub

**It's completely free!**

### Step 2: Navigate to Keys

Once logged in:

1. Click your profile icon (top right)
2. Select "Keys"
3. Click "Create Key"
4. Name it "AI Interview Practice"
5. Copy your key

### Step 3: Use Your Key

1. Visit the AI Mock Interview application
2. Paste your key in the login field
3. Click "Login"
4. Select a persona and start practicing

---

## Verify Your Key Format

Your key should look like this:

```text
sk-or-v1-xxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

**Important**: If it doesn't start with `sk-or-v1-`, you have the wrong key.

---

## Troubleshooting

### Invalid API Key Error

**Check:**

- Visit [OpenRouter.ai/keys](https://openrouter.ai/keys)
- Copy the entire key without extra spaces
- Verify it starts with `sk-or-v1-`
- Ensure the key is enabled (not disabled/revoked)

### Insufficient Credits

You've used your free tier credits.

**Solutions:**

- Check [OpenRouter Billing](https://openrouter.ai/account/billing/overview)
- Free tier includes starter credits
- Add a payment method for continued usage
- Request additional credits from OpenRouter support

### Rate Limited

You're sending requests too quickly (unlikely during normal interview).

**Solutions:**

- Wait a moment and try again
- Check your OpenRouter plan limits
- Consider upgrading plan for higher limits

## Security and Privacy

### Your Key is Safe

✅ Stored only in your browser (sessionStorage)
✅ Never sent to our servers
✅ Never logged or cached anywhere
✅ Automatically cleared when browser closes

### What You Should Know

⚠️ Treat your API key like a password
⚠️ Never share it publicly or on GitHub
⚠️ If compromised, regenerate immediately at openrouter.ai
⚠️ Monitor usage regularly to catch unauthorized access
